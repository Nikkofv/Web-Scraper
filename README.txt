This web is a sample web scraping tool written in Java which
Steps:  1. Retreive HTML webpage
        2. Extract desired sections from HTML page and save to file
        3. Parse through files and extract desired data
        4. Upload data into a table
        
 Table info creation and access directory:  \src\com\vogel\webscraper\copylib
 Executable Drivers and Tasks directory: \src\com\vogel\webscraper\lawyermi
 
 Executable Files with Main:
    SBMCheckInDriver.java:  execute steps 1 and 2
    SBMUSRFileParseDriver.jave:  execute steps 3 and 4
